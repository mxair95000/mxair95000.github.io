BoboTank 異星坦克：虛實結合的奪塔遊戲

設計簡介 Design Brief

Bobotank 異星坦克是款虛實結合的多人競技遊戲（MOBA）。兩隊共六位玩家操控實體坦克機器車，在充滿機關的場地中移動。在 15 分鐘時限內，以車載的雷射搶奪場地上的四座能量塔或擊倒敵軍的主堡坦克為目標，求得勝利。

遊玩時，玩家只能透過坦克車上的攝影機視角看到場地畫面，不能直接看到實體賽場，以此增加沉浸感與挑戰性；而觀眾可以從場地上方或其他視角的攝影機轉播畫面中看到玩家坦克們目前的狀態。

場地與硬體 Space & Hardware

場地大小為 5m x 7m ，機關設計左右對稱，各為紅藍兩隊的陣地。有許多會移動的機關，如移動牆、旋轉牆、翹翹板…等障礙物，四週的能量塔周圍都有擋牆，玩家可以善用地形和障礙物來閃躲敵人，跨過重重困難才能順利搶奪敵方的能量塔、保護隊友的主堡坦克。

遊戲規則 Game Rules

參賽者共六人，分為紅藍兩個隊伍，每隊其中一人擔任主堡坦克，其餘兩人擔任攻擊坦克。勝利方式為在 15 分鐘內佔領最多的能量塔，或將敵方的主堡坦克擊毀。若15分鐘時限已到尚未分出勝負，則會計算兩隊總剩餘血量，多者獲勝。




UI Flow｜介面流程

介面流程

遊戲進行前會先輸入競賽者暱稱和隊伍等資料料，進入後會有三秒開始倒數，隨即開始遊戲。在遊戲中若遭遇被擊重、隊友遭擊殺、坦克被搶都會有相對應的畫面反應。遊戲結束後除了告知輸贏，也有計分版告知玩家和觀眾各隊各玩家的詳細成績和表現。

UI Design｜介面設計

Wireframe 繪製過程


遊戲介面（可放大）

UI 發想的線框圖以紙本手稿為主；Mockup 使用 Sketch 設計，介面色調以黑和綠兩色為主凸顯遊戲趣味性。介面除了遊戲畫面設計，也包含有軟硬體設定、攝影機微調等畫面設計。
